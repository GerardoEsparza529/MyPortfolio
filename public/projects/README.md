# Imágenes de Proyectos

Coloca aquí las capturas de pantalla de tus proyectos con los siguientes nombres:

## Imágenes requeridas:

1. **darpha-app.jpg** - Darpha Maintenance App

   - Screenshot del dashboard o funcionalidad principal
   - Dimensiones recomendadas: 1200x800px o 16:9

2. **kontrolai.jpg** - KontrolAI

   - Screenshot del dashboard financiero o features principales
   - Dimensiones recomendadas: 1200x800px o 16:9

3. **accessai.jpg** - AccessAI

   - Screenshot del sistema de reconocimiento facial o dashboard
   - Dimensiones recomendadas: 1200x800px o 16:9

4. **chatbot-whatsapp.jpg** - WhatsApp AI Chatbot System
   - Screenshot del dashboard o conversaciones
   - Dimensiones recomendadas: 1200x800px o 16:9

## Foto de perfil (opcional):

Coloca tu foto de perfil en la carpeta `public/` con el nombre:

- **profile.jpg** o **profile.png**

Luego descomentar la línea en `src/sections/About.jsx`:

```jsx
<img src="/profile.jpg" alt={personalInfo.name} />
```

## Formato recomendado:

- Formato: JPG o PNG
- Calidad: Alta resolución pero optimizada para web (< 500KB por imagen)
- Aspecto: 16:9 para mejor visualización en cards
