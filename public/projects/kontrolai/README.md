3. **Presupuestos** - Vista de presupuestos y metas de ahorro
4. **Gráficas financieras** - Charts con Recharts/Chart.js
5. **Chat con IA** - Asistente virtual financiero en acción

8. **Proyecciones** - Proyecciones financieras generadas por IA

- Nombra los archivos descriptivamente: `dashboard.jpg`, `chat-ia.jpg`, etc.

- Calidad alta pero optimizada para web
