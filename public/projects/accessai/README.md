# AccessAI - Capturas de Pantalla

Agrega aquí todas las capturas de pantalla del proyecto AccessAI.

## Sugerencias de capturas:

1. **Dashboard de accesos** - Vista general del sistema
2. **Reconocimiento facial en acción** - Captura del proceso de verificación
3. **Historial de accesos** - Lista de entradas/salidas con timestamps
4. **Gestión de personas** - Administración de usuarios autorizados
5. **Estadísticas en tiempo real** - Métricas y gráficas con Recharts
6. **Control remoto** - Panel de control manual de puertas
7. **Vista del hardware** - Raspberry Pi con cámara y sensores
8. **Dashboard IoT** - Monitoreo de dispositivos conectados

## Formato:
- Nombra los archivos descriptivamente: `dashboard.jpg`, `facial-recognition.jpg`, etc.
- Formato: JPG o PNG
- Calidad alta pero optimizada para web
- Capturas con datos reales (o de prueba realistas)
